## keys

**Move the player**

Arrow left  : move left  
Arrow right : move right  
Arrow up : move forward  
Arrow down : move backward


**Move the camera**

A : move left  
D : move right  
W : move forward  
S : move backward


**Attack**

Space: attack  
x: throw something
p: drink portion

**Special**

o: orbit camara on or off for debugging  
f: show frames per second on or off
